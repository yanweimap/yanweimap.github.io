计算器

bc

bc 命令是命令行任意精度计算器语言，通常在linux下当计算器用。
它类似基本的计算器, 使用这个计算器可以做基本的数学运算。
支持加减乘除,指数(^),取余(%),开根(sqrt(n))等操作

支持参数

  1. -i 强制交互模式
  2. -l 指定标准数学库
  3. -w 给POSIX bc的扩展发出警告
  4. -s 准确处理POSIX bc语言
  5. -q 不要打印正常的GNU bc welcome

使用变量 ibase表示转换前数字的数制；obase表示转换后数字的数制(obase最好放在前面)

可以在bc中输入表达式,bc接受变量计算

  1. sqrt(n) 开根
  2. s(X) 正弦
  3. c(X) 余弦
  4. a(X) 反正弦
  5. l(N) 自然对数
  6. e(N) 指数对数
